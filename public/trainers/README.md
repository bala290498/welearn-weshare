# Trainer Images

This folder contains images for trainers/professional partners.

## Usage

Place trainer profile images in this folder. Images can be referenced in components using:

```tsx
<img src="/trainers/image-name.jpg" alt="Trainer Name" />
```

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Naming Convention

Use descriptive, lowercase filenames with hyphens:
- `balasubramani-arumugam.jpg`
- `trainer-name.jpg`
