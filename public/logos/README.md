# Company Logos

Place company logo images in this folder.

## Usage

Reference logos in your code using:
```
/logos/company-name.png
```

## Supported Formats
- PNG (recommended for logos with transparency)
- SVG (scalable vector graphics)
- JPG/JPEG (for logos without transparency)

## Example
- `/logos/google.png`
- `/logos/microsoft.svg`
- `/logos/amazon.png`
